## Задание 1







### Задача 1







### Текст задачи







#### На основе заданного списка чисел получить список из значений true и false. True
— если соответствующий элемент исходного списка чётный, и false — в
противном случае







### Алгоритм решения



######

##### Программа запрашивает количество чисел, собирает их в список через цикл, проверяет каждое число на чётность с помощью List.map и выводит полученный список из true/false на экран.

# Задание 2







###



### Текст задачи







#### Список содержит строки, обозначающие римские числа от I до IX. Найти в
десятичной системе счисления сумму элементов списка. 







### Алгоритм решения







#### Программа запрашивает количество чисел, циклически считывает римские строки в верхнем регистре в список, переводит каждый элемент в десятичное число и суммирует их с помощью List.fold, после чего выводит итоговую сумму.
