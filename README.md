# Задание 2







###



### Текст задачи







#### Список содержит строки, обозначающие римские числа от I до IX. Найти в
десятичной системе счисления сумму элементов списка. 







### Алгоритм решения







#### Ввод: Программа запрашивает у пользователя строку с римскими числами.Очистка: Переводит текст в нижний регистр и разбивает его по пробелам на отдельные слова.Конвертация: Превращает полученный массив строк в список F#.Подсчет: С помощью List.fold проходит по списку, сопоставляет каждое римское число (от i до ix) с арабским через функцию toInt и суммирует их. Нераспознанные знаки считаются за 0.
